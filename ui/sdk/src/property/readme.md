# Property

## 目标

减小每个Object对象的size，用于支撑窗口/页面中分配使用大量的object。


1. 每个属性要有一个唯一数字编码id，用于快速定位，减少使用map，能用数组定位就用数组定位。

2. 所有属性的默认值静态存储，共享。

3. 支持多优先级架构。 XML配置 > 继承 > 默认值

4. 可继承父结点属性，如字体属性。

5. 支持跨模块使用，每个模块可以定义自己的属性。


## 术语

* attribute

  是 xml/html 结点中的key/value键值对，是字符串类型

* property
  
  是attribute解析后的对象属性，可以是任意类型
